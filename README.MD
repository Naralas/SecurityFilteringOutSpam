## Setup

You need python installed and in your path (tested with 3.7)

### For linux based systems

```bash
virtualenv venv
source venv/Scripts/activate
pip install -r requirements.txt
jupyter notebook
```

### For Windows

```bash
virtualenv venv
venv/Scripts/activate/activate.bat
pip install -r requirements.txt
jupyter notebook
```
